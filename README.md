# switch-versions
Quick switching between different versions of binary sourced applications

## Applications support is initially intended for:
* packer
* terraform

## How to run:
* git clone https://github.com/laurence-hubbard/switch-versions.git
* cd switch-versions/
* ./init.sh
* switch-version
